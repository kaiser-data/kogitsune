## Takeaways for kit design

- **~115 skills (categories 1–7, 13) are the actual value on this machine.** The other
  ~160 — vertical playbooks, six unused language stacks, networking, social/marketing —
  are pure skill-list weight in every ECC session.
- ECC's own `agent-sort` skill exists precisely for this: it builds a DAILY vs LIBRARY
  install plan per repo. Worth running once as a second opinion on this categorization.
- If kogitsune's mirror gains per-skill (not just per-plugin) granularity for plugins,
  the natural kits fall straight out of the categories: `ecc-core` (1+2+3),
  `ecc-web` (+4+5), `ecc-py` (+6), `ecc-data` (+7), `ecc-research` (+13).
  Until then it's all-or-nothing via the `ecc` catalog entry.
